# MultiConnect 测试版安装与反馈说明

适用版本：v0.1.0 测试包

## 1. 安装

优先安装桌面上的测试安装包：

```text
MultiConnect_0.1.0_x64-setup.exe
```

Windows 可能提示“未知发布者”，这是因为当前测试版还没有做代码签名。确认来源可信后继续安装即可。

## 2. 首次配置

1. 打开 MultiConnect。
2. 进入左侧「设置」。
3. 在「OpenAI Key 管理」里填入自己的 OpenAI API Key。
4. 点击「测试连接」确认可用。
5. 添加 WhatsApp 账号，扫码登录。

注意：测试安装包不会内置开发者的 OpenAI Key。每台测试电脑都需要自己配置。

## 3. 重点测试项

请重点测试这些场景：

- 添加 WhatsApp 账号、扫码登录、关闭后重新打开。
- 多账号切换，尤其是 10 个以上账号时的顶部折叠和账号管理面板。
- 发送消息翻译：第一次点击发送按钮应只替换为译文，第二次点击才真正发送。
- 翻译失败时，中文原文不应被误发出去。
- 收到对方消息后，是否能看到中文翻译。
- 往上滚动查看历史消息时，历史消息翻译是否稳定。
- Ctrl+R 刷新后，界面和内嵌 WhatsApp 是否还保持正常。
- 打开「账号管理」后，WhatsApp 页面是否被正确遮罩，不应露出错位窗口。

## 4. 出问题时怎么反馈

请尽量同时提供：

1. 问题截图或录屏。
2. 发生问题的操作步骤，例如：
   - 打开客户端
   - 切换到第 3 个 WhatsApp 账号
   - 输入中文
   - 第一次点击发送
   - 出现错误提示
3. 进入「设置」→「测试诊断」→ 点击「导出诊断包」。
4. 把导出的 JSON 文件一起发回来。

诊断包不会包含真实 OpenAI Key，只会包含版本号、系统信息、路径、账号数量、Key 是否已配置等定位问题需要的信息。

## 5. 常见情况

### 翻译不可用

先检查：

- 设置页是否显示 OpenAI 已配置。
- 测试连接是否成功。
- 当前账号顶部翻译开关是否打开。

### 安装后打不开或空白

请截图，并导出诊断包。如果无法进入设置页，请说明 Windows 版本和安装方式。

### WhatsApp 要重新登录

测试版仍在完善 Session 保持能力，出现重新扫码时请记录：

- 是安装后第一次打开，还是重启后出现。
- 是单个账号掉线，还是所有账号掉线。
